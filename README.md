# Dotfiles setup

This repository contains configuration files for various tools, including Zsh, Tmux, and Neovim.

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/GasperKosenina/dotfiles.git ~/dotfiles

cd ~/dotfiles
```

### 2. Install GNU Stow

On macOS:

```bash
brew install stow
```

### 3. Apply the Configuration

```bash
stow zsh tmux nvim
```
